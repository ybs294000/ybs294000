## 🔨 What I'm Building

### [StudyDesk](https://github.com/ybs294000/StudyDesk) &nbsp;·&nbsp; Flutter · Dart · SQLite · Riverpod
> Offline-first study toolkit for Android and Windows — flashcards with SM-2 spaced repetition, timed quizzes, keyword-graded Q&A, Markdown notes with LaTeX, and local analytics. Runs as a portable Windows ZIP so students can carry their entire study system on a USB drive.

**Platform targets:** Android APK · Windows Portable · Linux · Web

---

### [SalaryScope](https://github.com/ybs294000/SalaryScope) &nbsp;·&nbsp; Python · Streamlit · Firebase · scikit-learn · XGBoost
> ML-powered career intelligence platform — salary prediction via manual input, resume upload (multilingual NLP), or bulk CSV. Five trained models including Random Forest (R²=0.9639) and XGBoost. Firebase auth, a schema-driven Model Hub for uploading custom models, HR compensation tools, financial planning suite, and an AI assistant.

**Live:** [salaryscope-app.streamlit.app](https://salaryscope-app.streamlit.app) · [Lite version](https://salaryscope-lite-app.streamlit.app)
